# Website Portofolio Pribadi - Muhammad Daffa Husen

Selamat datang di repositori kode sumber untuk website portofolio pribadi saya.

**🔗 Lihat Versi Live: [https://dappahsn.github.io/]**

## 👨‍💻 Tentang Website Ini

Website ini adalah **portofolio digital saya, Muhammad Daffa Husen**. Platform ini saya kembangkan untuk menampilkan perjalanan akademis, keahlian teknis, serta berbagai proyek yang telah dan sedang saya kerjakan di bidang teknologi dan rekayasa komputer.

Tujuan utama dari website portofolio ini adalah untuk:
* Menyajikan profil profesional saya secara ringkas dan menarik.
* Mendemonstrasikan kemampuan praktis dalam pengembangan web dan bidang terkait lainnya melalui contoh proyek nyata.
* Membangun kehadiran digital (digital presence) yang positif dan profesional.
* Menyediakan sarana bagi siapa saja yang ingin mengenal saya lebih jauh atau berdiskusi mengenai peluang kolaborasi dan profesional.

## 🚀 Teknologi yang Digunakan

Website ini dibangun dan dikelola menggunakan teknologi web berikut:

* **HTML5:** Untuk struktur dasar dan konten semantik dari halaman web.
* **CSS3:** Untuk styling, tata letak, dan aspek visual guna menciptakan tampilan yang menarik dan responsif.
* **JavaScript:** Untuk fungsionalitas interaktif, manipulasi DOM, dan peningkatan pengalaman pengguna.

## 🌐 Tujuan Repositori

Repositori GitHub ini berfungsi sebagai:
* Tempat penyimpanan utama untuk semua kode sumber dan aset website portofolio saya.
* Sistem kontrol versi untuk melacak setiap perubahan dan iterasi pengembangan.
* Platform untuk hosting website melalui GitHub Pages.

## 🛠️ Menjalankan Secara Lokal (Opsional)

Jika Anda ingin melihat atau melakukan modifikasi pada website ini secara lokal:
1.  **Clone repositori:**
    ```bash
    git clone [https://github.com/dappahsn/dappahsn.github.io.git](https://github.com/dappahsn/dappahsn.github.io.git)
    ```
2.  **Buka `index.html`:**
    Navigasi ke direktori hasil clone dan buka file `index.html` menggunakan browser web pilihan Anda.
